# ExpoApp Navigator 🚀

![GitHub last commit](https://img.shields.io/github/last-commit/KevinJG994/SimpleApp-ReactNativeExpo)

Proyecto de aplicación móvil desarrollada con React Native y Expo, implementando navegation y otras funcionalidades.

## Características

- 📱 Interfaz de usuario intuitiva y atractiva.
- 🚀 Implementación de navegación usando React Navigation.
- 💡 Uso de tecnologías modernas como Hooks y Context API.
- 🛠️ Personalizable y fácil de usar.
